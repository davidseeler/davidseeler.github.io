Gallery

Video Game Encyclopedia

Video game browsing application that promotes a simple and productive querying experience, presented in an elegant frontend developed with React.js. Game data is retrieved through API calls to RAWG.io and stored in a normalized MySQL database accessed through JDBC. Once data is collected, processed, and stored, a multi-threaded Java web server is enabled and returns serialized JSON game data to the calling frontend client.

Online Checkers

Online Checkers game developed with HTML, CSS, and vanilla JavaScript. Utilizes a multi-threaded Java web server to support multiplayer functionality. Also features a chat-box that displays System broadcasts and allows for messaging between the players and spectators.

Arcade App

Arcade application featuring my team's implementations of the games Space Invaders and 2048. With a focus on UX/UI in development, my partner and I aimed to deliver an engaging user experience with a design that emphasizes the classic-arcade retro feel. To push the theme even further, we were sure to include the iconic Hi-Scores page as well (records locally). Developed with Java, JavaFX, and CSS.

iTunes Gallery

Uses the iTunes API to retrieve and display images that correlate to key-words found in search query. Allows anything from artists, titles, genres, and even movies/shows to be searched. Results are randomly displayed and refreshed with a new image every second. Developed with Java, JavaFX, and CSS.

Portfolio

My portfolio, designed and developed by me. Uses particles.js, a custom menu button, and Font Awesome. Developed with Bootstrap, HTML, CSS, and JavaScript. Hosted with GitHub Pages.

Project-Uploader

When I began creating the gallery in my portfolio, I found that the process of uploading a gif for a project was much more tedious than I had expected. From having to deal with a buggy website to convert the screen recording into a gif, to downloading and refactoring files, to writing an entire block of HTML code and styling it -- the process took quite a bit of time. So, with the use of MoviePy, I wrote a Python script that instantly converts and uploads a gif to my gallery, sparing me the hassle of doing it manually.
